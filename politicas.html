⚙️ Configuración de Google Sheets

Sigue estos pasos una sola vez para activar el guardado en la nube

1

Crea una hoja de cálculo en Google Sheets

Ve a sheets.google.com, crea una hoja nueva y ponle el nombre "Aceptaciones Garantía". En la fila 1 escribe estos encabezados exactos (una columna por celda):

ID | Timestamp | Nombre | Correo | Telefono | TipoCliente | Aceptacion | FechaTexto
2

Crea el Apps Script

En la hoja de cálculo ve a Extensiones → Apps Script. Borra el código que aparece y pega el script que encontrarás en el archivo apps-script.gs incluido en este paquete. Luego haz clic en Guardar 💾.

3

Despliega el script como Web App

En Apps Script: Implementar → Nueva implementación. Configura así:

# Configuración de implementación: Tipo: Aplicación web Ejecutar como: Yo (tu cuenta de Google) Acceso: Cualquier usuario

Haz clic en Implementar, autoriza los permisos y copia la URL de la aplicación web que te entrega Google.

4

Pega la URL aquí para activar la conexión

Una vez tengas la URL del Web App, pégala aquí. Se guardará en este navegador.
